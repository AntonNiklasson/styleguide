Head (Enbart för utvecklare)

Head inkluderas längst upp på sidan för att sätta meta-data, referenser till resurser mm.

Kontext

Egenskap Beskrivning
title Sätter titeln på sidan. 
{
  "title": "Skolverket",
  "production": true
}
